Projects:

Code + more can be found on my github!

Netflix Updater

A Discord bot that provides updates on the newest Netflix releases, titles that will leave Netflix soon, the Top 10 Movies and TV shows of the day, and titles that will join Netflix soon. Built using JavaScript to scrape the data, Node.js, and Discord.js to connect the bot.

Animation Player/Editor

An animation player and editor that provides different ways to output an animation: textually, visually, or as an SVG file. Includes a way to play, pause, loop through the animation, and change the speed. Built using the Model-View-Controller design pattern in Java, and a GUI using Java Swing.

Aircraft Crashes Predictor

Created a Jupyter Notebook to analyze the data of aircraft crashes. Cleaned the data, trained and tuned various machine learning regression models to determine the best algorithm for predicting the number of fatalities in an aircraft using Python, pandas, and scikit-learn.
